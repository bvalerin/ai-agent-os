# Vision

## Proposito
Construir un sistema operativo ligero para proyectos asistidos por agentes de IA, donde roles, workflows, memoria y herramientas puedan coordinarse con trazabilidad.

## Resultado esperado
- Equipos humanos y agentes colaboran sobre una misma estructura.
- Las decisiones quedan registradas.
- La calidad y la operacion no dependen de memoria informal.

## Alcance inicial
- Definicion de agentes.
- Workflows tecnicos reutilizables.
- Memoria viva del proyecto.
- Integracion con herramientas via MCP.

## Fuera de alcance por ahora
- Elegir un vendor unico de LLM.
- Fijar un runtime unico para todos los proyectos.
- Resolver producto final mas alla del framework operativo.
