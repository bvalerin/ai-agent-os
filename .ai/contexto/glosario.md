# Glosario

- Agent OS: capa operativa del proyecto para coordinar agentes, memoria y workflows.
- Handoff: transferencia de contexto o responsabilidad entre agentes.
- ADR: Architecture Decision Record.
- MCP: Model Context Protocol, interfaz para conectar herramientas con agentes.
- Riesgo residual: riesgo que permanece despues de aplicar controles o pruebas.
