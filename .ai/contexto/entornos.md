# Entornos

## Local
Entorno principal durante bootstrap del proyecto. Debe permitir editar memoria, ejecutar scripts y validar workflows.

## CI
Entorno automatizado para lint, pruebas y chequeos de consistencia.

## Staging
Entorno opcional para validar integraciones y releases antes de produccion.

## Produccion
Pendiente de definir una vez exista una aplicacion desplegable.
