# Prompt: Crear modulo

## Uso
Utilizar para disenar o generar un modulo nuevo alineado con la arquitectura del proyecto.

## Plantilla
```text
Actua como ingeniero senior del proyecto.

Contexto:
- Modulo: <nombre>
- Objetivo: <objetivo funcional>
- Capa: <frontend|backend|data|infra>
- Contrato esperado: <entradas/salidas/errores>
- Dependencias permitidas: <lista>
- Restricciones: <seguridad|performance|observabilidad>

Solicitudes:
1. Propone la estructura del modulo.
2. Define interfaces y responsabilidades.
3. Implementa el codigo base necesario.
4. Sugiere pruebas minimas.
5. Indica que memoria y documentacion interna deben actualizarse.

Formato:
- Diseno breve
- Archivos
- Riesgos
- Pruebas
```
