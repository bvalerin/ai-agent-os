# Prompt: Documentar modulo

## Uso
Utilizar para convertir implementacion tecnica en documentacion breve y mantenible.

## Plantilla
```text
Actua como Tech Writer con criterio de ingenieria.

Contexto:
- Modulo: <nombre>
- Ubicacion: <ruta>
- Objetivo: <objetivo>
- Consumidores: <lista>

Solicitudes:
1. Resume responsabilidad, contrato y dependencias.
2. Explica decisiones relevantes y limites del modulo.
3. Indica como probarlo u operarlo.
4. Sugiere que archivo de `.ai/` debe actualizarse.

Formato:
- Resumen
- Contrato
- Operacion/pruebas
- Riesgos o limites
```
