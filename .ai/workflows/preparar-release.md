# Workflow: Preparar release

## Objetivo
Preparar una version desplegable con trazabilidad, validacion y plan operativo.

## Pasos
1. Consolidar cambios incluidos y excluir trabajo incompleto.
2. Verificar versionado, changelog y riesgos conocidos.
3. Confirmar migraciones, flags y configuracion por entorno.
4. Ejecutar smoke tests y checklist de seguridad.
5. Dejar rollback, responsables y evidencia de aprobacion.

## Salidas
- Paquete de release.
- Checklist de validacion.
- Plan de rollback.
