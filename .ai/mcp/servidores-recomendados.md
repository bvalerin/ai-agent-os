# Servidores MCP recomendados

## Basicos
- `filesystem`: lectura y escritura controlada del workspace.
- `git`: historial, diff y estado del repositorio.
- `browser`: automatizacion navegada y validacion UI.

## Utiles segun proyecto
- `docs`: consulta de documentacion externa.
- `database`: inspeccion segura de bases de datos.
- `secrets`: acceso controlado a credenciales del entorno.
- `observability`: consulta de logs, trazas y metricas.

## Criterios de seleccion
- Menor privilegio posible.
- Facilidad de auditoria.
- Valor real para workflows del proyecto.
