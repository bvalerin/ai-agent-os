# MCP

Esta carpeta centraliza la referencia operativa para integrar Model Context Protocol en el proyecto.

## Contenido
- `mcp.example.json`: ejemplo de configuracion inicial.
- `servidores-recomendados.md`: catalogo base de servidores MCP utiles para desarrollo.

## Uso esperado
1. Copiar `mcp.example.json` a la configuracion real del entorno.
2. Ajustar comandos, rutas y tokens segun la maquina o CI.
3. Registrar decisiones de adopcion en `../memoria/decisiones.md`.
